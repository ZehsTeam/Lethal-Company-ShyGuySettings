# ShyGuySettings
#### This mod lets you configure the Shy Guy (SCP-096) enemy from the Scopophobia mod. Probability Curve, Spawn Weight, Max Spawn Count, SpawnInside, and SpawnOutside.

#### <ins>YOU SHOULD USE THE [LETHALCONFIG](https://thunderstore.io/c/lethal-company/p/AinaVT/LethalConfig/) MOD TO EASILY EDIT THE CONFIG SETTINGS.</ins>

Download [ShyGuySettings](https://thunderstore.io/c/lethal-company/p/Zehs/ShyGuySettings/) on Thunderstore.

## Config Settings
> Note: You must load into the ship for all config settings to generate.

### ProbabilityCurve 
Determines how likely Shy Guy is to spawn throughout the day.

### SpawnWeight
Determines how often Shy Guy will spawn. Lower weights have a lower chance of spawning. Higher weights have a higher chance of spawning.

### MaxSpawnCount
The max amount of Shy Guys that can spawn.

### SpawnInside
If enabled, Shy Guy will be able to spawn inside.

### SpawnOutside
If enabled, Shy Guy will be able to spawn outside.
